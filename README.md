# plai-py
Play Minecraft with AI

## What is this
A port of OpenAI Video Pretraining for the real Minecraft.
It isn't a real mod and instead controls the mouse/keyboard for you.

## How to run
Install:

`pip install -r requirements.txt`

Run:

`python main.py`
